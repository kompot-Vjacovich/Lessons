Цвет
Свойство Описание
color Задает цвет текста.
opacity Определяет уровень прозрачности для элемента.
Фон и границы
Свойство Описание
background Позволяет установить все свойства фона в одном объявлении (универсальное свойство).
background-attachment Устанавливает, поведение фонового изображения во время прокрутки (фиксированное или прокручивается с остальной частью страницы).
background-color Определяет цвет фона элемента.
background-image Задает одно или несколько фоновых изображений для элемента.
background-position Задает положение (позицию) фонового изображения.
background-repeat Устанавливает, как будет повторяться фоновое изображение.
background-size Определяет размер фонового изображения/-ий.
border Позволяет установить все свойства границ в одном объявлении.
border-bottom Устанавливает все свойства нижней границы в одном объявлении. top - верхней, left - левой, right - правой.
border-color Устанавливает цвет для всех границ в одном объявлении.
border-bottom-color Устанавливает цвет нижней границы.
border-radius Позволяет определить форму границ элемента.
border-bottom-left-radius. Определяет форму границы нижнего левого угла.
border-style Устанавливает стиль для всех границ в одном объявлении.
border-bottom-style Устанавливает стиль нижней границы.
border-image Позволяет указать изображение, которое будет использовано вместо границы вокруг элемента. Свойство является универсальным и позоляет установить значения для всех border-image-* свойств в одном объявлении.
border-width Устанавливает ширину для всех границ в одном объявлении.
border-right-width Устанавливает ширину правой границы.
box-shadow Добавляет одну или несколько теней к элементу.
